Privacy Policy

Effective Date: 20 June 2025

Cloud Karamchari Todo List with Reminders is a Chrome extension that allows users to manage todo items and set local reminders. This extension respects your privacy and is built to function entirely within your browser.

1. Information Collection

This extension does not collect, transmit, or store any personally identifiable information (PII). All data, including tasks and reminder times, is stored locally in your browser using Chrome's storage API.

2. Use of Data

All tasks and reminders exist only on your local device. The extension does not send or share this data with any external server or third party.

3. Permissions

The extension uses only the following Chrome permissions to function:

  • storage: To store your todo list locally.
  • alarms: To trigger reminder notifications at specified times.
  • notifications: To display task reminders within Chrome.

4. Third-Party Access

No third-party services or APIs are used. All logic and data access is contained within the extension code.

5. Contact

If you have questions about this policy, you may contact the developer at: cloudkaramchari@gmail.com
